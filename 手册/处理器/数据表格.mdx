# 数据表格

<blockquote>
  <p>
    数据表格用来处理列表页面的表格布局
   
  </p>
  <p>
   `tables`表格中内置了许多处理器为我们提供完整的表格数据/布局处理，命名空间为
    <strong>table.&lt;表格名称&gt;</strong>；
  </p>

</blockquote>

数据表格相关处理器。

| 处理器                  | 说明                                     | 文档                          |
| ----------------------- | ---------------------------------------- | ----------------------------- |
| xiang.table.Search      | 按条件查询, 分页。                       | [查看](./process/Paginate)    |
| xiang.table.Find        | 按主键查询单条数据。                     | [查看](./process/Find)        |
| xiang.table.Select      | 返回数据集合，匹配选择器组件数据结构。   | [查看](./process/Get)         |
| xiang.table.Save        | 保存单条记录，指定主键更新，不指定创建。 | [查看](./process/Save)        |
| xiang.table.Delete      | 按主键删除单条记录。                     | [查看](./process/Delete)      |
| xiang.table.Insert      | 批量新增记录。                           | [查看](./process/Insert)      |
| xiang.table.UpdateWhere | 批量更新符合条件的记录。                 | [查看](./process/UpdateWhere) |
| xiang.table.DeleteWhere | 批量删除符合条件的记录。                 | [查看](./process/DeleteWhere) |
| xiang.table.QuickSave   | 保存多条记录，指定主键更新，不指定创建。 | [查看](./process/Save)        |
| xiang.table.UpdateIn    | 批量更新指定一组主键的的数据记录。       | [查看](./process/Update)      |
| xiang.table.DeleteIn    | 批量删除指定一组主键的的数据记录。       | [查看](./process/DeleteWhere) |
| xiang.table.Setting     | 读取数据表格配置信息, 用于前端界面渲染   | -                             |
