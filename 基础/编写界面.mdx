# 编写界面

<blockquote>
  YAO 内建 XGen 界面引擎，封装了 Table, Form, Chart
  等一系列常用功能，通过编写DSL即可实现多种功能。
</blockquote>

**约定**

1. 示例中约定应用根目录为 `/data/app`, 实际编写时需替换为应用根目录。
2. 使用 `<>` 标识自行替换的内容。 例如: `icon-<图标名称>`, 实际编写时应替换为: `icon-foo`, `icon-bar` ...

## 编写表格

### 添加表格 DSL 文件

在 `tables` 目录下, 创建一个 Table DSL 文件, 与一个 `product` 数据模型绑定。

在浏览器输入表格路由地址访问:

`http://<IP>:<YAO_PORT>/<管理后台路由前缀>/x/Table/product`

`/data/app/tables/product.tab.json`

```json
{
  "name": "产品",
  "action": {
    "bind": { "model": "product" }
  }
}
```

### 修改表格布局

修改 Table DSL, 刷新页面预览效果。 (如未开启 `watch` 需手动重启服务)

`/data/app/tables/product.tab.json`

```json
{
  "name": "产品",
  "action": { "bind": { "model": "product" } },
  "layout": {
    "table": {
      "columns": [
        { "name": "产品名称", "width": 200 },
        { "name": "上架状态", "width": 200 }
      ]
    }
  }
}
```

### 修改呈现组件

修改 Table DSL, 刷新页面预览效果。 (如未开启 `watch` 需手动重启服务)

`/data/app/tables/product.tab.json`

```json
{
  "name": "产品",
  "action": { "bind": { "model": "product" } },
  "layout": {
    "table": {
      "columns": [
        { "name": "产品名称", "width": 200 },
        { "name": "上架状态", "width": 200 }
      ]
    }
  },
  "fields": {
    "table": {
      "上架状态": {
        "bind": "online",
        "view": {
          "type": "Tag",
          "props": {
            "options": [
              { "label": "已上架", "value": "online" },
              { "label": "已下架", "value": "offline" }
            ]
          }
        },
        "edit": {
          "type": "Select",
          "props": {
            "options": [
              { "label": "已上架", "value": "online" },
              { "label": "已下架", "value": "offline" }
            ]
          }
        }
      }
    }
  }
}
```

## 表格关联表单

修改 Table DSL 文件，新增添加按钮、行编辑按钮。刷新页面预览效果。(如未开启 `watch` 需手动重启服务)

`/data/app/tables/product.tab.json`

```json
{
  "name": "产品",
  "action": { "bind": { "model": "product" } },
  "layout": {
    "filter": {
      "actions": [
        {
          "title": "添加产品",
          "icon": "icon-plus",
          "width": 3,
          "action": {
            "Common.openModal": {
              "Form": { "type": "edit", "model": "product" }
            }
          }
        }
      ]
    },
    "table": {
      "columns": [
        { "name": "产品名称", "width": 200 },
        { "name": "上架状态", "width": 200 }
      ],
      "operation": {
        "fold": false,
        "width": 255,
        "actions": [
          {
            "title": "查看",
            "icon": "icon-eye",
            "action": {
              "Common.openModal": {
                "Form": { "type": "view", "model": "product" }
              }
            }
          },
          {
            "title": "编辑",
            "icon": "icon-edit-2",
            "action": {
              "Common.openModal": {
                "Form": { "type": "edit", "model": "product" }
              }
            }
          }
        ]
      }
    }
  },
  "fields": {
    "table": {
      "上架状态": {
        "bind": "online",
        "view": {
          "type": "Tag",
          "props": {
            "options": [
              { "label": "已上架", "value": "online" },
              { "label": "已下架", "value": "offline" }
            ]
          }
        },
        "edit": {
          "type": "Select",
          "props": {
            "options": [
              { "label": "已上架", "value": "online" },
              { "label": "已下架", "value": "offline" }
            ]
          }
        }
      }
    }
  }
}
```
