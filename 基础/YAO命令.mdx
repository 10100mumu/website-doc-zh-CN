# YAO 命令

命令表:

| 名称            | 命令          | 参数                    | 说明                                                                 | 使用方法                                                                       |
| --------------- | ------------- | ----------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 执行脚本/数据流 | `yao run`     | `scripts/flows.xxx.xxx` | 运行一个处理器或者数据流                                             | 运行 scripts 目录下的 test.js 文件的 Search 方法 `yao run scripts.test.Search` |
| 数据迁移        | `yao migrate` | `-n`                    | 执行数据表的迁移功能,不加参数默认全部的数据表,加参数执行指定的数据表 | 迁移`models/pet.mod.json` 数据表,`yao migrate -n pet`                          |
| 启动 Yao        | `yao start`   |                         | 启动 Yao                                                             |                                                                                |
