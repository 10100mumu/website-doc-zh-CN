# 查询数据

<blockquote>
  <p>常规的数据CURD查询，可以使用数据模型的处理器实现。</p>
  <p>
    数据聚合统计等复杂的场景，或者需要对接数据仓库，可以在数据流中使用 Query DSL
    (Domain Specific Language) 实现。
  </p>
</blockquote>

## 使用数据模型处理器

编写数据流描述文件 `query.flow.json`，并放置在应用的 `flows` 目录中。

```json
{
  "label": "宠物",
  "version": "1.0.0",
  "description": "宠物统计",
  "nodes": [
    {
      "name": "列表",
      "process": "models.pet.Get",
      "args": [
        {
          "select": ["id", "name", "kind"],
          "wheres": [{ "column": "kind", "value": "猫" }],
          "limit": 3
        }
      ]
    },
    {
      "name": "分页",
      "process": "models.pet.Paginate",
      "args": [
        {
          "select": ["id", "name", "kind"],
          "wheres": [{ "column": "kind", "value": "猫" }],
          "limit": 1
        },
        1,
        3
      ]
    },
    {
      "name": "一条记录",
      "process": "models.pet.Find",
      "args": [1, { "select": ["id", "name", "kind"] }]
    },
    {
      "name": "更新记录",
      "process": "models.pet.Update",
      "args": [1, { "name": "这是一只懒猫" }]
    },
    {
      "name": "插入记录",
      "process": "models.pet.Create",
      "args": [
        { "sn": "300004", "name": "哮天犬", "type": "狗", "desc": "新成员" }
      ]
    },
    {
      "name": "删除记录",
      "process": "models.pet.Delete",
      "args": [1]
    }
  ],
  "output": "{{$res}}"
}
```

**运行数据流**

```bash
yao run flows.stat
```

## 使用 Query DSL 查询

Yao 提供以下 Query DSL, 可以在数据流节点中使用:

| DSL           | 说明                                                                 |
| ------------- | -------------------------------------------------------------------- |
| Gou Query DSL | 适用基于 SQL 的查询数据引擎。 <a href="#"> 查看文档 </a>             |
| Tai Query DSL | 适用基于 ElasticSearch 的数据引擎。<strong>(内测中暂未开放)</strong> |

编写数据流描述文件 `stat.flow.json`，并放置在应用的 `flows` 目录中。

```json
{
  "label": "宠物统计",
  "version": "1.0.0",
  "description": "宠物统计",
  "nodes": [
    {
      "name": "类型统计S1",
      "engine": "xiang",
      "query": {
        "select": [":COUNT(id) as 数量", "kind as 类型"],
        "groups": "类型",
        "from": "$pet"
      }
    },
    {
      "name": "类型统计S2",
      "engine": "xiang",
      "query": {
        "select": [":COUNT(id) as 数量", "kind as 类型"],
        "groups": "类型 ROLLUP 总计",
        "from": "$pet"
      }
    }
  ],
  "output": "{{$res}}"
}
```

| 字段   | 说明                                                |
| ------ | --------------------------------------------------- |
| engine | 数据引擎名称， `xiang` 使用数据模型存储的数据源连接 |
| query  | Query DSL，根据数据源连接类型，编写对应的查询逻辑   |

**运行数据流**

```bash
yao run flows.stat
```
